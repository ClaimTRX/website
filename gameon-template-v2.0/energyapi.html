Automate Energy Delegation with user.js

The user.js script allows you to delegate energy on the TRON blockchain, simplifying the process of renting energy for your transactions. Whether you're running a TRON-based project or need energy for smart contract interactions, this script automates the delegation process. You can choose to pay directly with TRX (sent automatically from your wallet) or use pre-deposited credits. The pricing is straightforward: 65,000 energy units for 6.5 TRX or 135,000 units for 13 TRX, for a 5-minute duration. This script is perfect for developers and users looking to streamline energy management on TRON.

Step-by-Step Guide to Using user.js

Follow these steps to set up and run the user.js script:

  1. Install Node.js: Ensure you have Node.js installed on your system. Download and install it from the official website if needed (nodejs.org).
  2. Create a Project Directory: Create a new directory for your project and navigate to it in your terminal. For example, on Linux or macOS, you can run: 
    mkdir energy-delegation && cd energy-delegation
  3. Install the tronweb Package: The script requires the tronweb library to interact with the TRON blockchain. Install it using npm by running: 
    npm install tronweb
  4. Copy the Script: Copy the user.js script below and save it as user.js in your project directory.
  5. Configure the Script: Open user.js in a text editor and update the following variables:
    • TRONGRID_API_KEY: Your TronGrid API key (get one here).
    • PRIVATE_KEY: Your TRON wallet private key (ensure this wallet has enough TRX for direct payments).
    • API_KEY: Your Energy API key (e.g., 77b9c4e8a2f1d6b3e9a0c8f7d4e2b1c9a3d5e6f7b8c9a0b1c2d3e4f5a6b7c8d9).
    • order.receiver: The TRON wallet address to receive the delegated energy.
    • USE_CREDITS: Set to true to pay with credits, or false to pay directly with TRX.
  6. Run the Script: Execute the script using Node.js by running: 
    node user.js

    The script will delegate energy to the specified wallet, either deducting credits or sending TRX automatically.

The user.js Script

Below is the complete user.js script. Copy it into a file named user.js in your project directory.

const TronWeb = require('tronweb').TronWeb;

// === CONFIGURATION ===
const TRONGRID_API_KEY = "YOUR-TRONGRID-API-KEY"; // Replace with your TronGrid API key
const PRIVATE_KEY = "YOUR-PRIVATE-KEY"; // Replace with your private key
const API_KEY = "YOU-API-KEY"; // Replace with your API key
const USE_CREDITS = false; // Set to true to use credits, false to pay directly with TRX
const PAYMENT_WALLET = "TRUnBRHsGVYeFuBccYac5wyWYBAgcnLzmn"; // DO NOT CHANGE

// === ORDER SETUP ===
const order = {
    receiver: "RECEVING WALLET", // Replace with the receiving wallet address
    resource_type: "energy",
    amount: 65000,  // Options: 65000 or 135000
    duration: 5     // Duration in minutes (DO NOT CHANGE)
};

// === PRICE LOOKUP ===
const PRICE_TABLE = {
    65000: 6.5,
    135000: 13
};
const trxPrice = PRICE_TABLE[order.amount];
if (!trxPrice) {
    console.error("❌ Invalid order amount. Must be 65000 or 135000.");
    process.exit(1);
}
const priceInSun = trxPrice * 1e6;

// === TRON SETUP ===
const tronWeb = new TronWeb({
    fullHost: 'https://api.trongrid.io',
    headers: { 'TRON-PRO-API-KEY': TRONGRID_API_KEY }
});
tronWeb.setPrivateKey(PRIVATE_KEY);
const senderAddress = tronWeb.address.fromPrivateKey(PRIVATE_KEY);

// === API SETUP ===
const url = "http://api2.cftecosystem.com/energy";
const headers = {
    "Content-Type": "application/json",
    "Api-Key": API_KEY
};

// === MAIN FUNCTION ===
(async () => {
    console.log(`🔑 Sender: ${senderAddress}`);
    console.log(`💸 Paying ${trxPrice} TRX for ${order.amount} energy (Duration: ${order.duration} min)`);
    console.log(`➡️ To vendor: ${PAYMENT_WALLET}`);
    console.log(`🎯 Energy receiver: ${order.receiver}`);

    if (USE_CREDITS) {
        const res = await fetch(url + "/create-order", {
            method: "POST",
            headers,
            body: JSON.stringify({ order, use_credits: true })
        });
        const result = await res.json();
        console.log(result);
        return;
    }

    try {
        // === Build & Sign TX ===
        const tx = await tronWeb.transactionBuilder.sendTrx(
            PAYMENT_WALLET,
            priceInSun,
            senderAddress
        );
        tx.feeLimit = 1_000_000;

        const signedTx = await tronWeb.trx.sign(tx, PRIVATE_KEY);

        // === Broadcast TX ===
        const broadcastRes = await tronWeb.trx.sendRawTransaction(signedTx);
        console.log("📡 Broadcast response:", broadcastRes);

        if (!broadcastRes.result) {
            console.error("❌ Failed to broadcast transaction.");
            return;
        }

        console.log(`✅ TX broadcasted: ${signedTx.txID}`);

        // === Call /create-order with full signed TX ===
        const res = await fetch(url + "/create-order", {
            method: "POST",
            headers,
            body: JSON.stringify({
                order,
                use_credits: false,
                signed_tx: signedTx // Required by backend
            })
        });

        const text = await res.text();
        let data;
        try {
            data = JSON.parse(text);
        } catch (err) {
            console.error("❌ Failed to parse response:");
            console.log(text);
            return;
        }

        if (!res.ok || data.success === false) {
            console.error("❌ Order creation failed:", data);
            return;
        }

        console.log("✅ Order created:", data);

        // === Poll for Delegation Status ===
        const orderId = data.data.order_id;
        const statusUrl = `${url}/delegation-status?requestId=${orderId}`;
        let attempts = 0;

        const interval = setInterval(async () => {
            attempts++;
            try {
                const statusRes = await fetch(statusUrl, { method: 'GET', headers });
                const statusData = await statusRes.json();
                console.log(`ℹ️ Status [${attempts}]:`, statusData);

                if (statusData.status === "delegated") {
                    console.log(`✅ Energy delegated: TxID ${statusData.txId}`);
                    clearInterval(interval);
                } else if (statusData.status === "failed" || attempts >= 60) {
                    console.log(`❌ Delegation failed or timed out.`);
                    clearInterval(interval);
                }
            } catch (err) {
                console.error(`⚠️ Polling error: ${err.message}`);
                if (attempts >= 60) clearInterval(interval);
            }
        }, 2000);

    } catch (err) {
        console.error("❌ Transaction error:", err.message);
    }
})();
