Rent Energy

Grow your wealth through staking rewards

The user.js script allows you to delegate energy on the TRON blockchain, making it easier to rent energy for your transactions without manual payments. You can choose to pay directly with TRX, where the script automatically sends the payment from your wallet, or use pre-deposited credits. The pricing is 65,000 energy units for 6.5 TRX or 135,000 units for 13 TRX, for a 5-minute duration. This script is ideal for automating energy delegation for your TRON-based projects.

Step-by-Step Guide

Follow these steps to set up and run the user.js script:

  1. Install Node.js: Download and install from nodejs.org.
  2. Create a Project Directory: 
    mkdir energy-delegation && cd energy-delegation
  3. Install the tronweb Package: 
    npm install tronweb
  4. Copy the Script: Save the provided script as user.js in your project folder.
  5. Configure the Script: Edit user.js and update:
    • TRONGRID_API_KEY: Get from Trongrid
    • PRIVATE_KEY: Your TRON wallet private key
    • API_KEY: Your Energy API key
    • order.receiver: Target TRON wallet
    • USE_CREDITS: Set true to use credits, or false to pay with TRX
  6. Run the Script: 
    node user.js

    The script will automatically delegate energy.
